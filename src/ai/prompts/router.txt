1) Identifique quais intenções estão presentes na mensagem do usuário, entre:
   - "alimentos" (registrar/substituir/excluir alimentos)

   - "plano" (criar plano de refeições)
   - "compras" (gerar lista de compras)
   - "visao" (mensagem inclui ou refere-se a foto do prato)
   - "dicas" (perguntas gerais de nutrição/treino, sem prescrição)
2) Separe a mensagem original em partes por intenção, quando aplicável.
3) Retorne apenas JSON válido no formato abaixo.

Saída JSON (somente JSON, sem texto extra):
{
  "intents": ["alimentos","exercicios"],
  "parts": {
    "alimentos": "texto que descreve alimentos/porções...",
    "exercicios": "texto que descreve treino..."
  },
  "confidence": { "alimentos": 0.95, "exercicios": 0.90 }
}
